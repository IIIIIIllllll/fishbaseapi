fishbase.ropensci.org/sealifebase

fishbase.ropensci.org/sealifebase is a RESTful Application Programming Interface (API) to the Sealifebase database.

Sealifebase database and web portal are completely separate entities from this API. Sealifebase is kind enough to give us a copy of their database, from which we provide some of it through this API.

Source code for this API: https://github.com/ropensci/fishbaseapi

API documentation

find the docs at https://fishbaseapi.readme.io/
